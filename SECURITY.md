# Security policy

Adaptive Remote is an alpha with physical-device output and sensitive capture permissions. Do not publish exploit details for an unpatched issue.

Report security concerns through GitHub's private vulnerability reporting for this repository, if enabled. If that option is unavailable, open a minimal issue asking the maintainers for a private reporting channel without including reproduction details.

High-priority reports include unintended output, failure to stop, connection to an unselected device, API-key exposure, capture without clear user action, sensitive data in logs, and exported Android components that accept untrusted commands.

Do not include real API keys, private screen captures, Bluetooth addresses, or other personal data in reports.
